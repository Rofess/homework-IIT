1. Установили git 
2. Зашли в папку, которая не находится под версионным контролем git. Выполнили команду git init создав в данной папке локальный репозиторий.
3. Cоздали файл "ot4et.txt" и выполнив команду git add ot4et.txt добавили его в локальный репозиторий.
4. Фиксируем проделанные шаги в отчете командой git add и создаем версию отчета в git.
5. Создаем новую ветку в git с помощью команды git branch и вносим в неё изменения отчета.
